Назначение программы:
Расчёт X эффективного по относительному давлению P/P0 в заданных точках поверхности.
Расчёт с минимальной автоматизацией. Для расчёта xeff сложных форм либо для случаев, когда необходима тотальная проверка точности расчёта.

Командная строка:
xeff.exe <имя файла исходных данных> <число Маха>

Формат входного файла:
В каждой строке до конца файла: таблица из трёх столбцов: 
<длина вдоль образующей> <Давление P/P0> <радиус миделя>
Описывает точки, по которым строится функция xeff. Точки выбираются таким
образом, чтобы между выбранными точками давление изменялось линейно.

Формат выходного файла:
<длина вдоль образующей> <Х эффективное, турбулентное в степени 0.2> <Х эффективное, ламинарное в степени 0.5>

Значение х эффективного выводится для точек, в которых задан коэффициент давления.
